CURSO ADMINISTRACIÓN DE BASES DE DATOS
COD. 204026


Objetivo general:

implementar y administrar una base de datos relacional sobre un sistema manejador de bases de datos conocido; como es MySQL; aplicando los conceptos y estructura del lenguaje estructurado de consultas SQL


Objetivo específico 1:
Crear una base de datos relacional utilizando un lenguaje definición de bases de datos LDD

Objetivo específico 2:
llevar a cabo las tareas de consulta o modificación de los datos contenidos en las Bases de Datos del Sistema Gestor de Bases de Datos; mediante el lenguaje de manipulación de bases de datos LMD.

Objetivo específico 3:
Controlar los procesamientos de transacciones en la base de datos Relacional, mediante el lenguaje de control de datos LCD.
